Crime Forecasting with Prophet

This project uses Meta's Prophet model to forecast crime patterns across different neighborhoods in Toledo, Ohio. The model incorporates seasonal trends and holiday effects to provide accurate predictions for law enforcement resource allocation.

Technical Details
  • Time series forecasting using Prophet model from Meta
  • R-based implementation with Shiny visualization dashboard
  • Census tract-level analysis of six major crime categories
  • Integration with demographic data for contextual analysis
Key Findings
  • Discovered weekly and monthly seasonality patterns in property crimes
  • Identified neighborhood-specific trends that correlate with demographic factors
  • Developed visualization techniques for communicating insights to stakeholders
